# Dokumentasi Project

Folder ini hanya menyimpan dokumentasi yang dipakai untuk Doktek, testing, dan traceability project TemanTumbuh.

## Dokumen Utama

| File | Kegunaan |
|---|---|
| `API_docs.md` | Referensi endpoint API resmi aplikasi. |
| `functional-testing.csv` | Daftar skenario pengujian fungsional untuk fitur MVP dan Non-MVP. |
| `unit-testing.csv` | Daftar skenario unit/service/API test yang berasal dari automated Vitest suite. |
| `non-functional-testing.md` | Ringkasan pengujian non-functional requirement dan bukti automated test. |
| `external-retrospective-think-aloud.md` | Template evaluasi eksternal Retrospective Think Aloud dengan Bug Report Section. |
| `requirement-traceability-matrix.md` | Matriks keterlacakan requirement ke endpoint/flow dan file test. |

## Catatan

- Dokumen hasil ekstraksi/antara yang duplikatif tidak disimpan di folder ini.
- Hasil lokal seperti coverage report, Playwright report, dan test results tidak masuk repository.
